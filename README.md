# 🏫데일리스쿨 : 쉽고 간단한 학교생활의 첫 출발

## 🚩프로젝트 소개

### 학교생활에 필요한 다용도 어플

- 시간표 정보를 받아와서 학급 시간표를 확인 가능하다.
- 오늘의 급식과 이번주 급식 정보를 볼 수 있다.
- 캘린더에 일정을 등록하여 일정을 확인할 수 있다.
- 홈 화면을 통하여 시험 디데이 및 오늘의 할 일을 확인할 수 있다.
